# AMORAL

AMORAL es un videojuego narrativo interactivo desarrollado como Trabajo Final de Grado (TFG).

El proyecto explora la depresión desde una perspectiva experiencial, con el objetivo de generar empatía a través de la interacción y la toma de decisiones.

## Descripción

El juego se desarrolla en un entorno reducido donde el jugador interactúa con distintos elementos del día a día del protagonista.

Cada decisión afecta al estado emocional del personaje, representado mediante un sistema de ánimo que influye en la experiencia de juego.

## Tecnologías

- Godot Engine
- GDScript

## Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio
2. Abrir Godot Engine
3. Importar el archivo `project.godot`
4. Ejecutar la escena principal

## Estado del proyecto

Versión en desarrollo correspondiente a la PEC 3 del TFG.
